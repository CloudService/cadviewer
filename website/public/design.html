Objective: (CAD translation service) Convert the CAD file from one format to a different one. It supports the most CAD formats.

User workflow
1. User selects a storage provider, such as box.com. The user agent (browser) will navigate to the authorization page of the box.com.
2. User inputs the account name and the password of the storage provider and grant the permission. The select file dialog appears.
3. User selects a file, such as wheel.step. The full path of the file will appear in the source file edit box, which is read only.
4. User Selects the destination file format, such as dwf. The full path of the destination file will appear in the destination file edit box, which is read only.
5. User inputs her/his email address.
6. User click the start button to start the translation process.
7. Once the translation is completed, the user receives the mail notification. And the destination file is uploaded to the user selected storage space.
